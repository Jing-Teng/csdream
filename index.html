Introduction

資夢

我們相信，資訊科技可以改變世界。寫程式是一件很酷的事。

《資夢》將帶領你接觸改變的工具，讓你學會打造屬於自己的未來。

這份教材將會介紹進入資訊領域時可能會有的課題、可以使用的資源，並連結網路上的文件，讓不熟路徑的旅人能快速找到可以瀏覽的地方。寫作基調是深入淺出簡介觀念，並盡可能吸引讀者興趣。

歷史

《資夢》的由來是 shaform 經營的一個網誌：《交資夢想》，一開始的目的是介紹資工的課程還有相關資源。除了希望能夠幫助學生學習以外，也希望讓高中生能夠提早認識資工系的課程，了解自己的興趣，也有機會提早學習。

現在的《資夢》不再以資工系為主要目標，而是希望成為帶領一般人踏入資訊科技的第一本書。

參與

《資夢》採用 gitbook 發佈。歡迎大家開立 issue 提供建議，也可以送 pull requests 來增加新內容。以下是可能有幫助的資源：
